# link on new tab
با استفاده از __blank میگوییم که لینک را در تبا جدید باز کن

```html
<html>
<head>
    <title>Page HTML</title>
</head>
<body>
<p align="center">
    <a href="http://google.com" سایت گوگل target="_blank"">
        لینک سایت
    </a>
    </p>
</body>
</html>
```
